- ✅ postgres (포트 5432)

### Edge Layer
- ✅ gateway (포트 8080) - Spring Cloud Gateway

### Domain Microservices
모든 서비스가 `../service.aiion.site`에서 빌드되도록 설정됨:
```yaml
calendar-service:
  build:
    context: ../service.aiion.site
    dockerfile: ./calendar-service/Dockerfile
```

### AI Services
모든 AI 서비스가 `../ai.aiion.site`에서 빌드되도록 설정됨:
```yaml
aihoyun-chatbot-service:
  build:
    context: ../ai.aiion.site
    dockerfile: rag/chatbot_service/Dockerfile
```

## ⚠️ 확인 사항

### 1. IDE 캐시 오류 (무해)
- `api.aiion.site/service/culture-service` 경로 참조 오류
- **원인**: IDE가 이전 경로를 캐시
- **영향**: 실제 빌드/실행에는 영향 없음
- **해결**: IDE 캐시 정리 또는 무시 가능

### 2. 환경 변수 확인
- ✅ `OPENAI_API_KEY` - `api.aiion.site/.env`에 설정됨
- ✅ `JWT_SECRET` - `api.aiion.site/.env`에 설정됨
- ✅ Redis 설정 - `SPRING_DATA_REDIS_HOST`, `SPRING_DATA_REDIS_PORT`

## 🚀 다음 단계

1. **전체 빌드 테스트**
   ```bash
   cd api.aiion.site
   docker-compose build
   ```

2. **서비스 실행 테스트**
   ```bash
   docker-compose up -d
   ```

3. **게이트웨이 연결 확인**
   - http://localhost:8080/actuator/gateway/routes
   - 각 서비스 엔드포인트 테스트

4. **프론트엔드 연결 확인**
   - http://localhost:3000 (www.aiion.site)
   - API Gateway (8080)를 통해 모든 서비스 접근
