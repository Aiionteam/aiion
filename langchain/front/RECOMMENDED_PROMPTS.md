# 추천 프롬프트 목록

RAG 시스템 테스트를 위한 추천 프롬프트입니다.

## 📚 기본 질문

1. **"LangChain이 뭐야?"**
   - LangChain 프레임워크에 대한 기본 설명 요청

2. **"RAG가 무엇인가요?"**
   - RAG (Retrieval-Augmented Generation) 기술 설명

3. **"pgvector의 역할은?"**
   - PostgreSQL pgvector 확장 기능 설명

4. **"OpenAI GPT 모델 설명"**
   - OpenAI의 GPT 모델에 대한 정보

5. **"벡터 데이터베이스란?"**
   - 벡터 데이터베이스의 개념과 활용

## 🔍 심화 질문

6. **"LangChain과 RAG를 어떻게 함께 사용하나요?"**
   - 두 기술의 통합 방법

7. **"pgvector를 사용한 벡터 검색은 어떻게 작동하나요?"**
   - 벡터 유사도 검색 원리

8. **"RAG 시스템의 장점은 무엇인가요?"**
   - RAG의 이점과 활용 사례

9. **"임베딩(Embedding)이란 무엇인가요?"**
   - 임베딩의 개념과 활용

10. **"LLM 애플리케이션을 구축하는 방법은?"**
    - LangChain을 활용한 LLM 앱 개발

## 💡 실용적 질문

11. **"FastAPI로 RAG API를 만드는 방법은?"**
    - API 서버 구축 가이드

12. **"PostgreSQL에서 벡터를 저장하는 방법은?"**
    - pgvector 사용법

13. **"OpenAI API를 사용한 임베딩 생성 방법은?"**
    - OpenAI Embeddings API 활용

14. **"문서를 벡터로 변환하는 과정은?"**
    - 문서 임베딩 프로세스

15. **"유사도 검색 결과를 개선하는 방법은?"**
    - 검색 품질 향상 팁

## 🎯 테스트용 질문

- **"LangChain이 뭐야?"** - 가장 기본적인 질문
- **"RAG가 무엇인가요?"** - 핵심 개념 질문
- **"pgvector의 역할은?"** - 기술적 질문
- **"OpenAI GPT 모델 설명"** - 서비스 관련 질문
- **"벡터 데이터베이스란?"** - 개념 질문

## 📝 사용 팁

1. **명확한 질문**: 구체적이고 명확한 질문이 더 좋은 답변을 얻습니다.
2. **한국어 사용**: 시스템이 한국어로 최적화되어 있습니다.
3. **컨텍스트 활용**: 검색된 문서를 바탕으로 답변이 생성됩니다.
4. **다양한 질문**: 다양한 주제로 테스트하여 시스템 성능을 확인하세요.
