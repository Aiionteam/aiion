# 감정 라벨 체계 변경 안내

## ✅ 변경 완료

감정 라벨 체계를 **3단계에서 6단계로 확장**했습니다.

---

## 📊 새로운 라벨 체계

### 기존 (3단계)
- `0`: 중립
- `1`: 긍정
- `2`: 부정

### 새로운 (6단계)
- `0`: 평가불가
- `1`: 완전긍정
- `2`: 긍정
- `3`: 평범
- `4`: 부정
- `5`: 완전부정

---

## 🔧 변경된 파일

### 1. `diary_emotion_schema.py`
- **emotion validation**: `[0, 1, 2]` → `[0, 1, 2, 3, 4, 5]`
- **라벨 설명**: 모든 라벨 매핑 업데이트
- **에러 메시지**: 새로운 라벨 체계 반영

### 2. `diary_emotion_service.py`
- **preprocess()**: 라벨 설명 업데이트
- **evaluate()**: 분류 보고서의 라벨 매핑 업데이트
- **predict()**: 예측 결과의 라벨 매핑 업데이트

### 3. `router.py`
- **API 설명**: 새로운 라벨 체계 반영

---

## 📝 CSV 파일 업데이트 필요

### 기존 CSV 파일의 라벨 변경

현재 CSV 파일에는 기존 라벨 값(0, 1, 2)이 있습니다. 새 라벨 체계에 맞게 변경해야 합니다.

#### 라벨 매핑 가이드

기존 데이터를 새 체계로 변환할 때 참고:

**기존 0 (중립) → 새로운 3 (평범)**
- 중립적인 내용은 "평범"으로 분류

**기존 1 (긍정) → 새로운 2 (긍정) 또는 1 (완전긍정)**
- 약간 긍정적: 2 (긍정)
- 매우 긍정적: 1 (완전긍정)

**기존 2 (부정) → 새로운 4 (부정) 또는 5 (완전부정)**
- 약간 부정적: 4 (부정)
- 매우 부정적: 5 (완전부정)

**새로운 0 (평가불가)**
- 감정을 판단하기 어려운 내용

## 🚀 다음 단계

### 1. CSV 파일 라벨 업데이트

`diary_emotion.csv` 파일을 열어서 각 행의 `emotion` 컬럼 값을 새 체계에 맞게 변경:

```csv
id,localdate,title,content,userId,emotion,
1,'1592-02-13,<임술>,"...",1,4,  # 부정 → 4 또는 5
2,'1592-02-14,<계해>,"...",1,3,  # 중립 → 3
...
```

### 2. 모델 재학습

라벨 변경 후 모델을 재학습:

```powershell
POST http://localhost:9003/diary-emotion/train
```

### 3. 예측 테스트

새 라벨 체계로 예측이 제대로 작동하는지 확인:

```json
POST http://localhost:9003/diary-emotion/predict
{
  "text": "오늘 정말 기분이 좋다!"
}
```

**예상 응답:**
```json
{
  "emotion": 1,
  "emotion_label": "완전긍정",
  "probabilities": {
    "평가불가": 0.05,
    "완전긍정": 0.85,
    "긍정": 0.08,
    "평범": 0.01,
    "부정": 0.005,
    "완전부정": 0.005
  }
}
```

## ⚠️ 주의사항

1. **데이터 양**: 6단계로 늘어났으므로, 각 라벨당 충분한 데이터가 필요합니다.
   - 최소 각 라벨당 10개 이상
   - 이상적으로는 각 라벨당 50개 이상

2. **데이터 균형**: 각 라벨의 데이터 분포를 균형있게 유지하는 것이 좋습니다.

3. **라벨 일관성**: 같은 내용에 대해 일관된 라벨링이 중요합니다.

## 📋 라벨 정의

### 0: 평가불가
- 감정을 판단하기 어려운 내용
- 예: "오늘 날씨가 맑다." (감정 표현 없음)

### 1: 완전긍정
- 매우 긍정적인 감정
- 예: "오늘 정말 기쁘고 행복한 하루였다!"

### 2: 긍정
- 긍정적인 감정
- 예: "오늘 기분이 좋다."

### 3: 평범
- 특별한 감정이 없는 일상
- 예: "오늘은 평범한 하루였다."

### 4: 부정
- 부정적인 감정
- 예: "오늘 좀 힘들었다."

### 5: 완전부정
- 매우 부정적인 감정
- 예: "오늘 정말 속상하고 슬프다."
